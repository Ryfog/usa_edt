# University of San Andreas — Emploi du temps

Gestionnaire d'emploi du temps (web) pour l'université RP. Classes ALPHA / OMEGA / SIGMA / DELTA,
matières colorées + professeurs, pauses REPAS / RÉCRÉATION.

## Tout gérer
Clique **✎ Modifier** :
- **＋ Jour** / **🗑 supprimer jour**, **＋ créneau** / **＋ pause**, **✕** pour retirer une ligne
- change les **heures**, **jours**, **dates**, **matières** (menu) — le **prof** suit la matière
- **🎨 Matières** : ajouter/éditer/supprimer une matière, sa **couleur**, son **icône**, son **prof**
- **📝 Textes** : titre, sous-titre, pied de page
- **⬇ Exporter / ⬆ Importer** : sauvegarde/transfert en JSON · **⟲ Réinitialiser**

Les modifications sont **sauvegardées automatiquement** dans ton navigateur (localStorage).
